# 🧑‍💼 User Management API (Node.js + SQLite)

## 📌 Features
- Create, retrieve, update, and delete users
- SQLite database integration
- UUID-based IDs
- PAN and mobile validation
- Manager ID verification
- Soft deletes

## 🛠 Tech Stack
- Node.js
- Express
- better-sqlite3
- UUID
- Thunder Client / Postman for API testing

## 📂 Setup

```bash
npm install
npm run dev
